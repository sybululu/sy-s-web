### ✅ Task 6: XSS防护 (P1)

**问题**: 用户输入可能包含恶意脚本

**修复方案**:
```python
@validator('text')
def validate_text(cls, v):
    dangerous_patterns = ['<script', 'javascript:', 'onerror=', 'onclick=']
    for pattern in dangerous_patterns:
        if pattern in v.lower():
            raise ValueError('输入包含非法字符')
    return v
```

**文件**: `python_backend/app_fixed.py`
